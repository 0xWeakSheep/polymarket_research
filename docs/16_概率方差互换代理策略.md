# 策略名称

概率方差互换代理策略

## 策略描述

这也是论文里的衍生品思路，但它关注的是概率本身 `p` 的方差，而不是 `logit` 的方差。

相较于 `logit` 方差，概率方差更直观，因为交易界面上看到的就是概率价格。  
不过从建模稳定性来说，它通常不如 `logit` 方差稳定。

## 策略举例

例子：

- 你预测未来 1 小时的概率方差 `Σ(Δp)^2` 很高
- 说明虽然方向未定，但价格大概率会来回大幅摆动
- 这时可以把它作为“高波动环境”的代理指标

## 策略获利方式

如果用代理回测的方式看，它赚的是：

- 实现的概率方差大于你预估的公平水平

如果用实战角度理解，它更适合支持：

- 区间突破策略
- 新闻前风险调整
- 中间区域波动交易

## 适用市场

- 更关注界面概率变化的研究
- 想把模型结果翻译成更容易解释的价格语言

## 基础开仓条件

- 预测未来 `Σ(Δp)^2` 明显偏高或偏低

## 基础平仓条件

- 预测窗口结束

## 风险提示

- 在接近 0 或 1 时，概率方差天然会被压缩
- 所以它更适合搭配 Swing Zone 使用
