# 策略名称

做市库存偏斜策略

## 策略描述

论文借用了做市领域很经典的思路：如果你的库存已经偏得太多，报价就不该继续保持完全对称。

简单说：

- 如果你手上 YES 太多，就应该更愿意卖出
- 如果你手上 YES 太少，就应该更愿意买入

这就是库存偏斜。

## 策略举例

例子 1：

- 你已经连续成交了很多买单，手里累积了大量 YES
- 这时继续按原价挂买单会让库存更偏
- 正确做法是把买价压低、卖价抬高，引导库存回到中性

例子 2：

- 你持仓偏空，而市场又进入高波动窗口
- 这时如果还不调报价，可能会在错误方向越积越多

## 策略获利方式

它主要赚的是“降低库存风险后的更稳定做市收益”。

更准确地说，它的作用是：

- 减少单边库存堆积
- 降低被单边行情拖累的风险
- 让做市收益更平滑

## 适用市场

- 自动挂单或做市型策略
- 需要持续管理库存的盘口环境

## 基础开仓条件

- 这类策略不是传统意义的开仓
- 而是在每次成交后，根据库存重新计算下一轮报价

## 基础平仓条件

- 当库存回到目标区间时恢复中性报价
- 极端库存时暂停一侧报价

## 风险提示

- 如果行情单边很强，库存偏斜也未必能完全挡住风险
- 数据颗粒度不足时，回测很容易过度理想化
