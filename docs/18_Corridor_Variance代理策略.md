# 策略名称

Corridor Variance 代理策略

## 策略描述

Corridor Variance 的意思是：不是统计整个样本的方差，而是只统计价格落在某个区间里时的方差。

论文很看重这个思路，因为在预测市场里，中间区域通常才是最有交易意义的区域。

对回测来说，这个策略可以改写成：

- 只研究 `p` 处于某个区间时的未来波动

## 策略举例

例子：

- 你定义 `p ∈ [0.35, 0.65]` 为核心交易区
- 只有当价格在这个区间里时，才统计未来方差
- 你再预测这个区间内的实现波动

如果这个指标比全市场方差更稳定，它会很适合做中间区域交易。

## 策略获利方式

从代理角度看，它赚的是：

- 你只在最值得交易的区间里做波动研究和风险配置

从实战角度看，它能帮助你：

- 把资金集中在高分歧区
- 避开靠近 0 和 1 的低质量波动

## 适用市场

- 高分歧的加密事件市场
- 想重点研究中间区域行为的策略框架

## 基础开仓条件

- 当前概率位于预设 corridor
- 同时模型预测该区间内未来波动活跃

## 基础平仓条件

- 离开 corridor
- 预测窗口结束

## 风险提示

- corridor 选得太窄，样本会不足
- corridor 选得太宽，又失去策略意义
